# JobConnect

## Sobre o Projeto

O JobConnect é uma plataforma simples de vagas inclusivas desenvolvida para a disciplina de Transformação Digital.

O objetivo do projeto é demonstrar como a tecnologia pode facilitar o acesso a oportunidades de emprego de forma simples, acessível e moderna.

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub
- Lighthouse

---

## Funcionalidades

- Exibição de vagas
- Interface responsiva
- Botões interativos
- Feedback ao usuário
- Layout moderno
- Avaliação de desempenho com Lighthouse

---

## Objetivo da Transformação Digital

O projeto busca demonstrar como soluções digitais podem melhorar processos de recrutamento e inclusão profissional, proporcionando mais acessibilidade e praticidade aos usuários.

---

## Resultados Obtidos

Durante o desenvolvimento foram realizadas melhorias de acessibilidade, SEO e boas práticas utilizando o Lighthouse do Google Chrome.

Pontuações obtidas:

- Accessibility: 100
- Best Practices: 96
- SEO: 90
